= 管理存储阈值
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以设置阈值、以便在卷和聚合容量达到特定级别时在vCenter Server中接收通知。

.步骤
. 登录到vSphere Client。
. 在快捷方式页面中，选择插件部分下的NetApp ONTAP tools*。
. 在ONTAP工具的左侧窗格中，转到 *设置* > *阈值设置* > *编辑*。
. 在“*编辑阈值*”窗口中，在“*接近满*”和“*满*”字段中提供所需的值，然后选择“*保存*”。您可以将阈值恢复为推荐的默认值：接近满格为 80，满格为 90。
