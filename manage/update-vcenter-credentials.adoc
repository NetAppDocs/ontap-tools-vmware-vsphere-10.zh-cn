= 更新 vCenter Server 凭据
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以使用维护控制台更新 vCenter Server 实例凭据。

.开始之前
您需要具有维护用户登录凭据。

.关于此任务
如果在部署后更改了 vCenter Server 凭据，请使用此过程更新它们。

.步骤
. 从 vCenter Server 中，打开一个 VASA Provider 控制台。
. 以维护用户身份登录。
. 输入 `2` 以选择System Configuration Menu。
. 进入 `8` 更改 vCenter 凭据。
