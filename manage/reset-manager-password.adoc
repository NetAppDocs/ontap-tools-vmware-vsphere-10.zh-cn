= 重置ONTAP工具管理器密码
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
如果您忘记了ONTAP工具管理器密码，可以使用从ONTAP tools for VMware vSphere维护控制台生成的重置令牌来恢复管理员访问权限。

.步骤
. 打开网页浏览器并导航至 `https://<ONTAPtoolsIP>:8443/virtualization/ui/`访问ONTAP工具管理器。
. 在登录页面，选择“重置密码”。
. 使用ONTAP tools for VMware vSphere维护控制台生成密码重置令牌：
+
.. 登录 vCenter Server 并打开维护控制台。
.. 进入 `2`选择“系统配置”。
.. 进入 `3`选择“更改‘maintenance’用户密码”。


. 在密码重置对话框中，输入重置令牌、用户名和新密码。
. 选择“重置”以更新凭据。
. 使用新密码登录ONTAP工具管理器。
