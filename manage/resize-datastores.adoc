= 调整NFS和VMFS数据存储库的大小
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
通过调整数据存储库大小、可以增加虚拟机文件的存储。您可以根据基础架构要求的变化更改数据存储库的大小。

.关于此任务
您可以增加 NFS 和 VMFS 数据存储的大小。这些数据存储中的FlexVol volume不能缩小到低于其当前大小，但可以增长到 120%。

.步骤
. 在vSphere Client主页中、选择*主机和集群*。
. 在导航窗格中、选择包含数据存储库的数据中心。
. 右键单击NFS或VMFS数据存储库，然后选择*NFS NetApp ONTAP tools*>*Resize DataDataStore。
. 在“调整大小”对话框中，输入数据存储的新大小，然后选择“确定”。
