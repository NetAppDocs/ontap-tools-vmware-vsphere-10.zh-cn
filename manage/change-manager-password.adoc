= 更改ONTAP工具管理器密码
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以使用ONTAP工具管理器更改管理员密码。

.步骤
. 从Web浏览器启动ONTAP工具管理器： `\https://<ONTAPtoolsIP>:8443/virtualization/ui/`
. 使用您的ONTAP tools for VMware vSphere管理员凭据。
. 选择屏幕右上角的*管理员*图标，然后选择*更改密码*。
. 在更改密码弹出窗口中，输入旧密码和新密码。用户界面屏幕会显示密码要求。
. 选择“更改”以应用更改。
