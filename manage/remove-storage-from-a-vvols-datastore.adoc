= 缩减数据存储库
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
本节介绍如何从vVols数据存储中删除卷。

您可以对 vCenter Server 中ONTAP tools for VMware vSphere发现或管理的任何vVols数据存储区使用从数据存储区中删除存储的操作。

如果卷包含vVols，则无法从卷中删除存储；对于此类卷，删除选项将被禁用。从数据存储中删除卷时，您还可以选择从ONTAP存储中删除选定的卷。


NOTE: 基于ASA r2 系统的vVols数据存储不支持缩小vVols数据存储操作。

.步骤
. 在vSphere Client主页中、选择*主机和集群*。
. 在导航窗格中、选择包含数据存储库的数据中心。
. 右键单击VVOR数据存储库、然后选择* NetApp ONTAP tools*>*从数据存储库中删除存储*。
. 选择没有vVols的卷并选择*删除*。
+

NOTE: 选择vVols所在卷的选项已被禁用。

. 在*Remove storage*弹出窗口中，选中*Delete volumes" from ONTAP cluster-*复选框以从数据存储库和ONTAP存储中删除卷，然后选择*Delete*。
