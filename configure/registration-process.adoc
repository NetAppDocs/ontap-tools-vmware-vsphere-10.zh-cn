= 使用ONTAP工具向 vCenter Server 实例注册 VASA 提供程序
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
使用ONTAP tools for VMware vSphere将 VASA 提供程序注册到 vCenter Server 实例。这使得基于存储策略的管理、 vVols支持以及与ONTAP系统上的 VMware Live Site Recovery 设备的集成成为可能。

VASA 提供程序设置显示所选 vCenter Server 的注册状态。

.步骤
. 登录到vSphere Client。
. 选择插件部分下的*快捷方式*>* NetApp ONTAP tools*。
. 选择“设置”>“VASA 提供程序设置”。  ONTAP工具显示 VASA 提供程序注册状态为未注册。
. 选择*注册*按钮以注册VASA Provider。
. 输入 VASA 提供程序的名称和凭据。用户名只能包含字母、数字和下划线。将密码长度设置为 8 到 256 个字符之间。
. 选择 * 注册 * 。
. 成功注册并刷新页面后， ONTAP工具将显示已注册的 VASA 提供程序的状态、名称和版本。


.下一步行动
验证已启用的VASA Provider是否列在vCenter Client的VASA Provider下：

.步骤
. 转到 vCenter Server 实例。
. 使用管理员凭据登录。
. 选择*存储提供程序*>*配置*。验证是否已正确列出已启用的VASA Provider。
