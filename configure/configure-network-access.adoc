= 配置网络访问
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
除非已配置网络访问，否则从 ESXi 主机发现的所有 IP 地址都会自动包含在导出策略中。您可以配置策略以仅允许某些 IP 地址并排除其他 IP 地址；但是，如果您尝试从排除的 ESXi 主机挂载，则操作将失败。

.步骤
. 登录到vSphere Client。
. 在“插件”部分的“快捷方式”页面中选择NetApp ONTAP tools*。
. 在ONTAP工具的左侧窗格中，转到 *设置* > *管理网络访问* > *编辑*。
+
要添加多个IP地址、请使用逗号、范围、无类别域间路由(CIDR)或这三者的组合分隔列表。

. 选择 * 保存 * 。
