= 适用于VMware vSphere的ONTAP工具概述
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
ONTAP tools for VMware vSphere是一套用于虚拟机生命周期管理的工具。它与 VMware 生态系统集成，以简化数据存储配置并为虚拟机提供基本保护。它是作为开放虚拟设备 (OVA) 部署的水平可扩展、事件驱动的微服务的集合。

ONTAP tools for VMware vSphere支持：

* 核心虚拟机 (VM) 功能，例如保护和灾难恢复
* 用于基于存储策略的管理的 VASA 提供程序
* 基于存储策略的管理
* Storage Replication Adapter （ SRA ）


.适用于 VMware 的ONTAP工具的高可用性
ONTAP tools for VMware vSphere提供高可用性 (HA) 支持，有助于在发生故障时维持不间断运行。

HA 解决方案可帮助您从以下类型的中断中快速恢复：

* 主机故障 - 仅支持单节点故障。
* 网络故障
* 虚拟机（客户操作系统）故障
* 应用程序（ONTAP工具）故障


您无需执行任何其他配置即可ONTAP tools for VMware vSphere。

[NOTE]
====
ONTAP tools for VMware vSphere不支持 vCenter HA。

====
要使用 HA 功能，请确保在部署期间或稍后在 VM 设置中启用 CPU 热添加和内存热插拔。
